#### checkip
This is a small project in C that helps to learn and calculate different IP ranges, wildcards, and subnet-masks.

#### Usage
If checkip is ran without any input, it gives the different modes that you can use as response.
```
usage:
-i flag to enter interactive mode
-s to show dotted-decimal notation version of /24 style subnetmask 
<subnet mask> to show the corresponding wildcard
<ip address> <subnet-mask> information about IP address and subnet combination
<ip address> <ip address> <subnet-mask> check if IPs are in the same subnet
``
